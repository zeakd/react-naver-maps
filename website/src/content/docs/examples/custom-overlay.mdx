## 설명

`CustomOverlay`는 `position` 좌표에 `children`을 렌더링합니다. React의 `createPortal`을 사용하므로 일반 React 컴포넌트를 그대로 활용할 수 있습니다.

스타일은 CSS로 자유롭게 지정하세요.
