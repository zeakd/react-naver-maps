> [네이버 공식 튜토리얼 - 지도 경계](https://navermaps.github.io/maps.js.ncp/docs/tutorial-4-map-bounds.example.html)

## 설명

`ref`로 map 인스턴스에 접근하여 `fitBounds()`를 호출합니다. `onBoundsChanged` 이벤트로 현재 경계를 추적하여 `Rectangle`로 표시합니다.
