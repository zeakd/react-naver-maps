## 설명

`zoomControl={false}`로 기본 줌 컨트롤을 숨기고, 지도 컨테이너 위에 절대 위치 버튼을 배치합니다. `ref`로 map 인스턴스에 접근하여 `setZoom()`을 호출합니다.
