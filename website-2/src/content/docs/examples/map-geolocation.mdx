## 설명

`navigator.geolocation.getCurrentPosition()`으로 현재 위치를 가져온 뒤, `ref`를 통해 `map.setCenter()`와 `map.setZoom()`을 호출합니다. `InfoWindow`로 위치 정보를 표시합니다.
