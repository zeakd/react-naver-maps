`InfoWindow` 컴포넌트는 정보 창을 표시합니다. 문자열 또는 React children을 content로 받습니다.

## 사용법

```tsx
import { InfoWindow } from 'react-naver-maps';

<InfoWindow content="안녕하세요" position={{ lat: 37.5, lng: 127.0 }} />;
```
