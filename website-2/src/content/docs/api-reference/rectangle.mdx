`Rectangle` 컴포넌트는 지도 위에 사각형을 그립니다.

## 사용법

```tsx
import { Rectangle } from 'react-naver-maps';

<Rectangle bounds={{ south: 37.5, west: 126.9, north: 37.6, east: 127.0 }} />;
```
